<h1 align="center">Vue2 + TSX + Rspack</h1>

## 技术栈
- Vue2
- JSX / TSX
- Rspack
- Less

## 快速开始
```bash
# 安装依赖
yarn install

# 启动服务
yarn dev

# 构建
yarn build
```
